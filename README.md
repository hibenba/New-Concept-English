# 📘 新概念英语 New Concept English Notes (NCE 1-4)

A structured PDF note collection for **New Concept English**, organized into four levels:
一份针对《新概念英语》的结构化 PDF 记笔记集合，共分为四个级别。
- 📗 nce1 https://www.ncego.com/books/nce1
- 📘 nce2 https://www.ncego.com/books/nce2
- 📙 nce3 https://www.ncego.com/books/nce3
- 📕 nce4 https://www.ncego.com/books/nce4

All notes are carefully handwritten and organized for learning, review, and printing.
所有的笔记都是精心手工编写的，并且经过了整理，便于学习、复习和打印使用。

## 小小说明

本人是一个待业在家的互联网从业者，非专业的程序员，之所以要折腾这个站。是因为看到太多劣质课程坑钱，自己作为一个中年被动学习者（笔记写来是给自己孩子使用的）走过太多弯路。
《新概念英语》来华几十年以来，我甚至翻遍了整个互联网都没有找到一份好用的完整笔记，有的事情是需要有人来做的，那个人是我又何尝不可呢？
经历了多次内容迭代，也非常感谢通过“错误举报”纠结一些语法错误、单词与内容拼写错误的同学，在未来我相信这份笔记会越来越好，大家一起加油~

## ✨ Features 特色

- Vocabulary breakdown with phonetics 词汇表
- Yes/No Question structure explanations 语法解析
- Grammar comparison tables 语法结构表
- Sentence transformation exercises 句子转换及练习
- Natural phonics introduction 自然拼读
- Dolch Basic Word List extension 必备的词汇表
- Idioms and practice questions 每个pdf都有不同的习语内容
- Printable layout design 专为打印优化
